Comandos de git mas utilizdos

Solo debe de existir una etiqueta h1 pero las etiquetas h2,h3...h6 pueden existir las que nosotros queramos

  • git status sirve para verificar el estatus de nuestros archivos de nuestro proyecto
  • git add -A sirve para agregar los archivos que ya fueron creados
  • git commit -m sirve para agregar un comentario
  • git config --global user.name sirve para agregar o editar un nombre de usuario
  • git config --global user.email sirve para agregar o editar un correo
  • git config --global --list sirve para enlistar los elementos del usuario
  • git branchnos muestra las ramas que tenemos
  • git branch nombre_de_la_ramasirve para crear una rama nueva
  • git checkout nombre_de_la_ramasirve para movernos entre las ramas
  • git status nos muestra si hemos eliminado o modificado un archivo nuevo
  • git lognos muestra los commits que tenemos
  • git merge nombre_de_la_ramafusionamos la rama con master,tenemos que estar posicionado en master
  • git branch -d nombre_de_la_ramaNos ayuda a eliminar una rama
  • git remove add origin link_de_githubSirve para conectar mi proyecto con mi repositorio de github
  • git remove -vSirve para berificar mi conexion de mi proyecto con mi repositorio de github
  • git push origin NombreDeLaRama Subes tus commits a tu repositorio de GITHUB
  • git pull origin NombreDeLaRama Descargas los avances de cualquier rama que tenga mi repositorio de GITHUB, normalmente se descargan de la rama master de mi GITHUB git pull origin master
  • git tag Sirve para ver las versiones que tenemos
  • git tag -a v1.0 -m "NombreDeLaVersion" Sirve para nombrar y crear versiones que en un futuro podre descargar
  • git push origin --tag Sirve para subir los tags que alla creado
