Comandos de CSS

  • color Para cambiar el color de las palabras
  • background Para cambiar el color de fondo
  • background-colorPara cambiar el color de fondo de un bloque
  • boderGenerar un borde a un sección
  • displayNos ayuda a pasar block a flex para que podamos manipular una sección
  • font-sizeCambia el tamaño de letra
  • font-familyCambia el tipo de letra
  • justy-contentNos ayuda a cambiar de lugar un bloque
  • aling-itemsCambia de lugar el contenido de un bloque
  • flex-directionCambia la posiscion de que se encuentra algunos bloques
  • font-weightCambia el grosor de las palabras
  • font-styleAyuda cambiar el estilo de letra a italica
  • padding-inline-start Sirve para poner quitar un espacio al inicio de un texto
  • display:flex Sirve para trabajar con flexbox, me coloca cada contenedor uno a lado del otro de manera horizontal, para poder utilizar FLEXBOX siempre debo utilizar display:flex
  • text-decorationSirve para cambiar la decoracion del texto para desabilitar en una particular
  • :hoverSirve para que realize un cambio o una transicion y esto se le puede configurar dandole propiedades "transicion"
  • transicionEs una propiedad que se le asigna a :hover y sirve para darle el numero de segundo para que realize el cambio
